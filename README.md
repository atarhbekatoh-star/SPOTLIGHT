# Project: SPOTLIGHT
We are building an interactive learning app that helps people boost their confidence, public speaking skills, and become more comfortable expressing themselves all through fun, structure, and consistent practice. Think of it like Duolingo, but for social confidence. Our goal is to turn confidence into a learnable skill, not a personality trait.
CORE FEATURES
Daily Missions:One challenge every day to act like a "MAIN CHARACTER".
Sreaks: Keep your fire burning by completing tasks daily.
The Path:Level up from "Extra" to "Leading role".
